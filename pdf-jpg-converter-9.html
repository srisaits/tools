Understanding Quality Settings

Image Quality: Determines the compression level of the JPG output. Higher values mean better quality but larger file sizes.

DPI (Dots Per Inch): What is DPI? DPI stands for Dots Per Inch and determines the resolution of the output image. Higher DPI means more detail but larger file size. Affects the resolution of the output image. Higher DPI is better for printing, while lower DPI is sufficient for screen viewing.

Tips for Best Results

  • For printing, use 300 DPI or higher
  • For web use, 72-150 DPI is usually sufficient
  • Adjust quality between 70-90% for a good balance of quality and file size
  • If converting many pages, consider downloading as a ZIP file
